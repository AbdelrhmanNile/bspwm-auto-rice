## bspwm auto rice [bspar]
an automatice ricing script for bspwm (my dots)

ONLY WORKS ON ARCH BASED DISTROS

## imgur video demo
<img>https://i.imgur.com/VBXp4yK.mp4</img>

## Installation
```pip install bspwm-auto-rice```

## Usage
for help:
```bspar --help```
and 
```bspar <command> --help```
example:
```bspar set --help```
